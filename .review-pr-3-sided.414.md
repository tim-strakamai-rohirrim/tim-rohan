### Out-of-diff observations (dependency `arc_agent_writer/factories/acquisition_requirements.py` — verified on branch, NOT in this PR)
These are excluded from the verdict but this PR newly exposes them via conf-controlled `document_uris`:
- **[security/major] SSRF** — `factory:79` `_ALLOWED_URI_SCHEMES = {"http","https","s3","minio"}`. A caller controlling `document_uris` can supply `http://169.254.169.254/...` or any internal URL and `stage_documents` will fetch it. **Recommend:** drop `http`/`https` (restrict to `s3`/`minio`) or hostname-allowlist before GA.
- **[security/major] path-traversal** — `factory:226-228` the `load_bytes(run_id, uri)` fallback passes the full external URI as the run-relative `path` arg; if a backend resolves it relatively, `../../other_org/...` could cross tenant prefixes. **Recommend:** remove the fallback (no shipped backend exposes the cross-prefix getter per the docstring) or assert the org-scoped prefix.
- **[robustness/major] run_meta persistence** — `factory:162` `run_meta["documents"] = documents` mutates in-memory only; confirm the orchestrator checkpoints `ctx.manifest` before the step returns, else a mid-run restart loses the staged-document list.
